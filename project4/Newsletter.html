But that's not all! We've also been working on some new features that we're excited to share with you:

Change Request Templates

Creating change requests can be time-consuming, especially when you're dealing with similar changes over and over again. To help streamline this process, we've added the ability to create templates for common change requests. This means that you can quickly create a new change request based on a template and save time by not having to re-enter the same information every time.

Request Dashboards

We know that tracking change requests can be a challenge, especially when you're dealing with a high volume of requests. To help make this process easier, we've added new dashboards that provide an at-a-glance view of all your change requests. You can see how many requests are in progress, how many have been approved, and how many are waiting for approval. You can also filter and sort the dashboard to see the information that's most relevant to you.
